A clock for the Radio 4 Today Programme

Famously, presenters of Radio 4's Today programme occasionally have trouble telling the time. Although this endears them to regular listeners, the clock below is designed to help. Alongside an optional traditional analogue clock, a special clock is presented that spells out the time in the style of spoken English (actually British English, since our American cousins do it differently).

Technical

To incorporate in a web page simply include the today-clock.js script and a div element with class today-tell-time. If you also want a regular analogue clock then include a second div with class today-clock. On load, the script populates any div that has the today-tell-time class with a 'tell time' clock and populates any div that has the today-clock class with an analogue clock. Both clocks auto-size to fill the containing div (even dynamically if the containing div is resizable). However, the analogue clock will always maintain a 1:1 aspect ratio no matter what the aspect ratio of its containing div. The 'tell time' clock will stretch to match the size of its container div but looks best with an aspect ratio of 3:1.

Uses the HTML5 canvas element. Should work in any modern web browser.

Accuracy: The clock simply echoes that of the client machine. Where ultimate precision is required the computer's clock should be synchronised with a reliable time server, such as ntp2d.mcc.ac.uk
